# SmartKeyboard
SmartKeyboard is a lightweight, intelligent input method manager designed specifically for macOS users who frequently switch between different languages. Tired of constantly changing your input method when moving between your code editor, web browser, and chat applications? SmartKeyboard solves this frustrating problem automatically.

### Key Features:

• **Automatic App-Switching**: SmartKeyboard detects which application is currently active and automatically switches to your preferred input method for that specific app.

• **Menu Bar Quick Access**: Easily configure default input methods for any active window through the convenient menu bar icon.

• **Smart Analytics**: Analyzes your input method switching habits and intelligently anticipates when to switch, saving you time and effort.

• **Ad-Free Experience**: Clean, lightweight design with no advertisements or unnecessary bloat.

• **Privacy-Focused**: All data processing happens locally on your Mac - your typing habits never leave your device.

### Perfect For:

• Developers who frequently switch between English code and native language documentation/comments
• Multilingual professionals working with translation tools and communication apps
• Content creators who write in multiple languages across different platforms
• Anyone who uses different languages for different applications

### How It Works:

1. SmartKeyboard runs quietly in your menu bar
2. When you switch applications, it automatically detects the active app
3. Your pre-configured input method for that app is instantly applied
4. The app learns from your habits and becomes smarter over time

Stop wasting time with manual input method switching. Let SmartKeyboard handle the heavy lifting so you can focus on what matters most - your work and communication.
